ClaudeSwitch

A desktop client for Claude AI that lets you seamlessly switch between your Claude.ai subscription and Anthropic API credits — with local conversation history and multi-account support.

Download View on GitHub

Seamless Switching

Toggle between Claude.ai subscription and API credits mid-session without losing context.

Local Storage

All conversations stored in SQLite on your device. Nothing leaves your machine.

Multi-Account

Manage multiple Claude accounts in one dropdown. Switch instantly and carry context over.

Import from Claude.ai

Bring your existing Claude.ai conversations into the desktop client with one click.

Artifact Export

Export any conversation as a clean HTML artifact you can share or archive.
