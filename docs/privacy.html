Privacy Policy

Last updated: June 27, 2026

Short version: ClaudeSwitch stores everything locally on your device. We do not collect, transmit, or sell any personal data. Your conversations never leave your machine unless you explicitly export them.

1. What ClaudeSwitch Is

ClaudeSwitch is a free, open-source desktop application that provides a graphical interface for interacting with Claude AI. It is not affiliated with Anthropic. The source code is publicly available at github.com/i348v/claudeswitch under the MIT License.

2. Data We Do Not Collect

ClaudeSwitch does not operate any servers. There is no backend, no analytics, and no telemetry. We do not collect:

  • Your name, email address, or any personal identifiers
  • Your conversation history or message content
  • Your API keys or authentication credentials
  • Usage statistics, crash reports, or diagnostic data
  • Device information or IP addresses

3. Data Stored Locally on Your Device

ClaudeSwitch stores the following data only on your local machine, never transmitted to us:

  • Conversations — stored in a SQLite database at ~/.claude_client/conversations.db
  • Account configuration — stored in ~/.claude_client/config.json, including account labels, mode (subscription or API), and API keys you enter
  • Exported artifacts — HTML files saved to ~/claude_artifacts/

You can delete any of these files at any time to remove all local data.

4. Third-Party Services

ClaudeSwitch communicates directly with third-party services on your behalf:

  • Anthropic API — if you use API Credits mode, your messages are sent to Anthropic's API. See Anthropic's Privacy Policy.
  • Claude Code CLI — if you use Subscription mode, messages are processed via the Claude Code CLI on your device, which uses your Claude.ai session. See Anthropic's Privacy Policy.
  • Google Gmail API (optional) — if you use the "Import from Claude.ai" feature with Google Sign-In, ClaudeSwitch requests read-only access to your Gmail inbox solely to detect and download your Anthropic data-export email. We request the minimum scope (gmail.readonly). Your Gmail credentials and messages are never stored by ClaudeSwitch beyond the duration of the import session.

5. Google API Scopes

When you use the Gmail import feature, ClaudeSwitch requests the following Google OAuth scope:

  • https://www.googleapis.com/auth/gmail.readonly — read-only access to detect the Anthropic export email and extract the download link

ClaudeSwitch's use of information received from Google APIs adheres to the Google API Services User Data Policy, including the Limited Use requirements.

6. Children's Privacy

ClaudeSwitch is not directed at children under 13. We do not knowingly collect data from children.

7. Changes to This Policy

If this policy changes materially, the updated version will be posted at this URL with a new "Last updated" date. Continued use of the application after changes constitutes acceptance.
